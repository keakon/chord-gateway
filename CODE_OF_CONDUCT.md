# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/) Code of Conduct.

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Standards

Positive behavior includes:

- Respectful communication
- Constructive feedback
- Responsibility for mistakes

Unacceptable behavior includes:

- Personal attacks, insults, harassment
- Publishing private information without explicit permission

## Enforcement

Maintainers are responsible for clarifying and enforcing these standards.

## Reporting

Please report abusive behavior privately to maintainers.
For security issues, use [SECURITY.md](./SECURITY.md).
