# 行为准则

本项目遵循 [Contributor Covenant](https://www.contributor-covenant.org/) 行为准则。

## 我们的承诺

我们承诺为所有参与者提供无骚扰的协作环境。

## 行为标准

鼓励的行为：

- 尊重沟通
- 建设性反馈
- 对错误负责并及时修正

不可接受的行为：

- 人身攻击、侮辱、骚扰
- 未经授权公开他人隐私信息

## 执行

维护者负责解释并执行本准则。

## 反馈渠道

如遇不当行为，请私下联系维护者。
安全问题请参见 [SECURITY.md](./SECURITY.md)。
